FLONAR is designed and programmed in the backyard

Mit FLONAR kann geprüft werden, ob Hecken in allen 26 Kanton in der Schweiz die erlaubte Maximalhöhe überschreiten oder nicht. Dies unter Berücksichtigung des jeweiligen Grenzabstands. Eine technologische Unterstützung zur Klärung von Nachbarschaftsstreitigkeiten im Zusammenhang mit Hecken.
